# Шаблоны Revit add-in

Файлы в папках `Revit2023` и `Revit2024` являются шаблонами для разработки и документации.
Маркер `__SABPLUS_INSTALL_DIRECTORY__` заменяется установщиком на фактический локальный путь.

Готовый `SABPlus.Radial.Setup.exe` сам создаёт рабочие `.addin`-файлы в:

- `%AppData%\Autodesk\Revit\Addins\2023`
- `%AppData%\Autodesk\Revit\Addins\2024`
