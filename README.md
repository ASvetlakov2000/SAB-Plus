# SAB+ — Revit Productivity Tools

SAB+ — локальная экосистема инструментов для Autodesk Revit. В текущей версии реализуется первый модуль: радиальное колесо команд.

## Радиальное колесо

- `XButton1` (ближняя боковая кнопка мыши) открывает кольцо стадий.
- `XButton2` (дальняя боковая кнопка) открывает команды активной стадии.
- Активная стадия хранится отдельно для каждого проекта Revit.
- Overlay работает только на переднем плане Revit и только в открытой проектной модели.
- Команды Revit передаются через локальный bridge и выполняются штатно через `UIApplication.PostCommand`.
- Системные команды используют точные имена `PostableCommand`, а интерфейс показывает русские названия из отдельного словаря.

## Установка

1. Соберите Release-пакет командой:

   ```powershell
   .\build\Build-Release.ps1 -RevitApiDir "C:\Path\To\RevitAPI"
   ```

2. Запустите `artifacts\SABPlus.Radial.Setup.exe`.
3. Выберите Revit 2023 и/или Revit 2024.
4. После установки перезапустите Revit.

Установка выполняется для текущего пользователя и не требует прав администратора. Пользовательские настройки находятся в `%LocalAppData%\SAB+\RadialWheel` и сохраняются при удалении приложения.

## Сборка

- Visual Studio 2022 или актуальный .NET SDK с поддержкой .NET Framework 4.8.
- Autodesk Revit API 2023 (`RevitAPI.dll` и `RevitAPIUI.dll`). Используемые API совместимы с целевыми Revit 2023–2024.
- C# 7.3, WPF, Newtonsoft.Json 13.0.3.

Проекты:

- `SABPlus.Radial.Core` — модели, JSON, миграции и геометрия.
- `SABPlus.RevitBridge` — Revit add-in, `ExternalEvent`, проверка и `PostCommand`.
- `SABPlus.Radial.Overlay` — глобальные триггеры, tray, overlay и редактор.
- `SABPlus.Radial.Setup` — автономный установщик с встроенным payload.
- `SABPlus.Radial.Core.SmokeTests` — smoke-тесты геометрии, миграций и ограничений контекста.

## Ограничения текущей версии

- В словарь включаются только проверенные `PostableCommand` с подготовленными русскими названиями.
- Обучение сторонним add-in по журналу Revit будет добавлено отдельным этапом.
- Остальные модули SAB+ (режимы отображения, локальная библиотека и 3D Cutter) пока не реализуются.
